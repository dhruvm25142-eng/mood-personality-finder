MoodScope – Mood-Based Personality Finder
📌 Purpose

MoodScope is a web-based application designed to analyze a user’s mood or personality based on their responses to simple questions. The goal of this project is to provide users with personalized insights and suggestions while demonstrating the use of API integration and interactive UI design.

🌐 API Used

This project uses the DummyJSON API to fetch a collection of quotes.

The application selects a random quote from the dataset and displays it based on the user’s detected mood.

🚀 Features (Planned)
🧠 Analyze user responses to determine mood/personality (Happy, Sad, Overthinker)
💬 Display dynamic quotes based on mood
🔄 Generate a new quote on demand
❤️ Save favorite quotes using local storage
🌙 Toggle between dark mode and light mode
📱 Fully responsive design for mobile, tablet, and desktop

🛠️ Technologies Used
HTML
CSS
JavaScript
React.js
Fetch API

▶️ How to Run the Project

Clone the repository:

git clone https://github.com/dhruvm25142-eng/mood-personality-finder.git

Navigate to the project folder:

cd moodscope

Install dependencies:

npm install

Start the development server:

npm start

Open in your browser:

http://localhost:3000
